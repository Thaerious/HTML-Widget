NAME
    nidget - manage nidget based projects

SYNOPSIS
    nidget [OPTION]... [COMMAND]

OPTIONS
    -n, --name
        Set the name of the nidget when relevent

    -v, -vv
        Display verbose information, or very-verbose information

    --dist
        Set the 'distribution' directory for the package command

    --output
        Set location for local output files

COMMANDS
    readme:  print the readme file which contains an example setup
    init:    create a new nidget project in the this directory
    create:  create a new nidget with specified name (-n)
    view:    create a new view with specified name (-n)
    records: show all records
    clean:   remove --dist and --output directories and remove records from nidget.json file

    babel:   translate .mjs files into .js placing them in the --dist directory, adds/updates records in nidget.json
    sass:    render .css files placing them in the --dist directory, adds/updates records in nidget.json
    pack:    browserify .js files for all the views, place in --output directory
    ejs:     render .ejs files to .html, place in --output directory [1]

NOTES
    [1] the ejs command will copy .css & .mjs to --output
